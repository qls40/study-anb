## 증감연산자 ++와 ?가 변수의 앞에 위치할 때와 뒤에 위치할 때의 
## 차이를 확인할 수 있는 코드를 작성합니다.

<br><br>


++x	먼저 피연산자의 값을 1 증가시킨 후에 해당 연산을 진행함.<br>
x++	먼저 해당 연산을 수행하고 나서, 피연산자의 값을 1 증가시킴.<br>
--x	먼저 피연산자의 값을 1 감소시킨 후에 해당 연산을 진행함.<br>
x--	먼저 해당 연산을 수행하고 나서, 피연산자의 값을 1 감소시킴.<br><br><br>
```
void main() {
    int i = 1;
    int j = ++i;
    printf("i:%d, j:%d", i, j);
    }

출력 : i : 2, j :2;
```
```
void main() {
    int i = 1;
    int j = i++;
    printf("i:%d, j:%d", i, j);
}

출력 : i : 2, j : 1
```
